# Разведка: русскоязычная сцена 2024–2026

Кандидаты в пантеон (§1), новые приёмы в копилку (§5, §11, §14), обновления blend-карты (§24).

## 1. Главные тренды сцены 2024–2026

### 1.1 Саундклауд-волна (СК-фрешмены)
- Кто: **Madk1d** («Sexyswag» — альбом года по The Flow), **Темный Принц** («Мрачные треды», главный фрешмен 2025), **Урал Гайсин** («Саундклауд пак»), Фортуна 812.
- Суть: лоу-фай, анти-полировка, среда передаётся через бытовые «крючки»: голосовое знакомого, мамино предостережение про энергетики, неймдроппинг актуальных брендов и артистов.
- Что берём в пайплайн: приём «крючок среды» (см. §3.1 ниже).

### 1.2 Новая народность / фолк-хоррор
- Кто: **ATL** («Необратимые последствия»: мир «за забором ПО-2», фолк-хоррор + электроника), **Букер** («Время колокольчиков» — прямая отсылка к Башлачёву), **Бонд с кнопкой** («Дом»: школа АукцЫона/Колибри, виральные «Камушки», «Кухни», «Сидр»).
- Суть: архаика, причет, духовный стих, деревенская и окраинная оптика без лубка; фолклорные формулы внутри современного бита.
- Это самый перспективный кандидат на новый жанровый плейбук → см. `genres/folk_horror.md`.

### 1.3 Лирика умолчаний (эзопов язык военного времени)
- Кто: **Хаски** («Партизан»: без прямых нападок, но с посвящением погибшему товарищу; «Ветром снегом зноем»), **Metox** («Лойко Зобар»: автор как «пишущая машина» внутри системы), **НОМ** («Сволочь»: мрачная сатира, юродство).
- Суть: сказать больше, не называя; умолчание и иносказание как художественный ресурс, а не только цензурная необходимость.

### 1.4 Антигламурная женская оптика
- Кто: **Размаха** («Р96»: «брутальная, антиняшная, хулиганская» женственность, провинциальная самобытность + личный лор), **Sky Rae** («Gonki»: панчлайны типа «Хочешь сына? Могу предоставить»), **polnalyubvi** (личная трагедия → пронзительная лирика).
- Суть: женский голос без кукольности — расширяет галерею голосов §7.

### 1.5 Рэп-искренность, сделанная неуютной
- Кто: **Basic Boy** («Никто, нигде»: открытая зависть к другим как стержень альбома), **Иван Дорн** («Dead Blogger»: расставание с амбициями, зацикленное финальное четверостишье).
- Суть: не «мне больно», а некрасивые эмоции: зависть, мелочность, тщеславие. Прямое усиление нашего §9 «уродливое»: уродливой бывает не только фактура, но и эмоция.

### 1.6 Вокал как инструмент, телесная психоделия
- Кто: **Gruppa Skryptonite** («04:00 p.m.»: питч, дробление, шёпот, «лагутенковское мяуканье»), **Джон Гарик** («Престиж»: цельная история через абстрактные ванлайны).
- Суть: текст пишется под вокальную пластику — аргумент за новый параметр P11 singability в функции оценки 2.0.

## 2. Кандидаты в пантеон (дополнение к §1)

### ATL — фолк-хоррор окраин
- Что брать: мифологизация панельного быта («забор ПО-2» как мифологема), бестиарий («Ящер», «Паразиты мозга»), сдержанная надежда внутри мрака.
- Чего избегать: прямое копирование его шаманской интонации без его бестиария — получится косплей.

### Джон Гарик — абстрактный ванлайн с интертекстом
- Что брать: сращение классики и сленга в одной строке: «У лукоморья дуб зелёный, gold chain на дубе том»; цельная история из разрозненных ванлайнов.
- Чего избегать: абстракция без сквозного сюжета рассыпается в шум.

### Madk1d — крючки среды
- Что брать: документальные детали поколения (голосовые, энергетики, неймдроппинг) как якоря времени.
- Чего избегать: эпатаж ради эпатажа без среды — быстро стареет.

### Букер — башлачёвская линия
- Что брать: колокольчиковая образность, работа с наследием русского рок-бардовского мифа без кавер-позиции.

### Бонд с кнопкой — аукцыоновская глоссолалия в TikTok-формате
- Что брать: фолклорная ритмика + бытовая конкретика («Кухни», «Сидр»); доказательство, что «не для всех» масштабируется до арен.

### Размаха — антиняшный женский лор
- Что брать: провинциальная самобытность как капитал, личный лор-вселенная в текстах.

### Basic Boy — неуютная искренность
- Что брать: некрасивые эмоции от первого лица без самооправдания — новый подвид голоса «исповедник» (§7): **исповедник-завистник**.

## 3. Новые приёмы в копилку

### 3.1 Крючок среды (расширение §5: якорь/триггер)
```
ПРИЁМ: вместо общего описания среды — один документальный артефакт эпохи
ПРИМЕРЫ: недослушанное голосовое; банка энергетика на подоконнике;
  чек из Пятёрочки в кармане куртки; разряженный самокат поперёк тротуара
ПРАВИЛО: артефакт должен датировать сцену без указания года
АНТИ-СЛОП: это прямой антидот к vague_deixis и genitive_metaphor
```

### 3.2 Сращение классики и сленга (расширение §11: интертекст)
```
ПРИЁМ: хрестоматийная цитата ломается современным словом внутри строки
ЭТАЛОН: "У лукоморья дуб зелёный, gold chain на дубе том" (Джон Гарик)
ПРАВИЛО: цитата должна быть узнаваема с первых слов, слом — неожидан,
  но семантически мотивирован (цепь = златая цепь у Пушкина)
```

### 3.3 Зацикленный финал (расширение §4: структура)
```
ПРИЁМ: финальное четверостишье повторяется 2–4 раза без изменений,
  пока смысл не начнёт стираться/меняться от повтора (эффект мантры/выгорания)
ЭТАЛОН: Иван Дорн "Dead Blogger"
КОГДА: темы навязчивости, выгорания, расставания с иллюзией
НЕ ПУТАТЬ с chorus_checklist: здесь повтор буквальный и осознанный, а не вариации шаблона
```

### 3.4 Умолчание как приём (расширение §14: тёмная лирика)
```
ПРИЁМ: главное событие не называется — описываются только его следы в быту
ПРИМЕР МЕХАНИКИ: не "он погиб", а "его номер теперь просто цифры в телефоне"
КОГДА: темы потери, страха, цензурируемой реальности
БОНУС: умолчание автоматически повышает P1 concreteness и убивает position_explanation
```

### 3.5 Исповедник-завистник (новый подвид голоса, §7)
```
ГОЛОС: герой признаётся в некрасивом (зависть, мелочность, злорадство)
  без самооправдания и без финального искупления
ЭТАЛОН: Basic Boy "Никто, нигде"
ОПАСНОСТЬ: скатиться в саможалость — тогда это уже не уродливая эмоция, а кокетство
```

## 4. Предложения по blend-карте (§24)

- Зелёные новые: **ATL + Хаски** (общая мифологизация окраин, разные темпераменты); **Бонд с кнопкой + АукцЫон** (прямое наследование); **Размаха + Кровосток** (бытовой натурализм, разный гендерный ракурс).
- Красные новые: **Madk1d + Pyrokinesis** (стилистически несовместимые словари: СК-быт vs книжная патетика); **Букер + Пошлая Молли** (бардовская серьёзность vs трэш-ирония).
