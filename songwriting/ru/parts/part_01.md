## Принципы

Три правила, на которых стоит всё остальное:

1. **Звук — в большинстве консультаций.** Конкретный, составной. «саб на 60 Гц гудит, а на 80 — провал», не «бас плохой». В лирике — запах и фактура как якорь реальности.
2. **Один бьющий образ лучше трёх красивых.** Конкретные существительные и сильные глаголы. Абстракции — враги.
3. **Считай ритм вслух, не гадай.** Ритм — это не «ощущение». Это удары. Тоника — удары, не слоги.

Всё дальше — раскрытие этих трёх.

## 1. Пантеон

Расширенный список авторов, по которым калибруется «живой» почерк. По каждому — что брать.

| Автор | Жанр | Что брать |
|---|---|---|
| **Хаски** | рэп → тонический стих | бытовое уродство, регистровый контраст, фонетика как структура, «сумасшедшие сказки», телесный код |
| **Pyrokinesis** | рэп-рок/альт | ризомная образность, литературные отсылки, концептуальность, диалог с классикой |
| **Shortparis** | пост-панк/авангард | верлибр, рефрены-якоря, вокализации, тревога через образы-триггеры |
| **Скриптонит** | хип-хоп | уличный сленг, бытовая наблюдательность, социальный комментарий без пафоса |
| **ЛСП** | рэп/нуар | нуарная нарративность, гедонизм→смерть→воскресение, сленг+литература |
| **Аффинаж** | пост-панк/инди | образность, меланхолия, гармония стиля и бунта |
| **ploho / моль / сруб** | пост-панк/думер | холодный быт, мёртвый тон, разреженность, панелька как декорация |
| **Буерак** | пост-панк (сибирский) | абсурдно-ироничные тексты, мем-культура, соцкоммент через подъёб, бытовой гротеск |
| **Кровосток** | абстрактный хип-хоп | сатира, чёрный юмор, холодная констатация насилия как быта, аллитерация |
| **Saluki** | хип-хоп/эксперимент | меланхолия, коллажность, продюсерская цельность, интертекстуальность |
| **Пошлая Молли** | поп-панк/синтпанк | ирония над шоу-бизом, банальные рифмы как приём, вирусная прямолинейность |
| **25/17** | рэп/социальный | нарратив, исповедь, фолк-рэп, «культурка» как разговор с поколением |
| **Oxxxymiron** | рэп/литература | интертекстуальность, сложная метафора, диалог с классикой (Горгород→Пушкин) |
| **Сергей Бабкин** | фолк/акустик | интимность, тихая сила, разговорный тон, пауза как музыка |
| **Земфира** | рок/поп-рок | сжатая образность, короткие строки, метафора без объяснений |
| **Гражданская Оборона** | панк | сырой крик, политическая лирика, абсурд как протест |
| **АукцЫон** | рок/авангард | потоки сознания, вокализации, абсурдная образность |
| **Макулатура** | пост-панк/spoken word | декламация, литературность, холодный тон |
| **Чёрная Речка / Ploho-наследие** | пост-панк | унаследованная холодность, панелька, быт как конец |
