## 6. Синтаксис лирики

Синтаксис — как слова соединяются. В лирике синтаксис несёт ритм не меньше, чем метрика.

### 6.1 Приёмы

**Инверсия** — нарушение прямого порядка слов. «Солнце тает в окне, как харчок золота» — не «как золотой харчок». Инверсия сдвигает ударение на нужное слово.

**Парцелляция** — разрыв предложения на части. «ты пришла. в три четырнадцать. и осталась.» Каждая часть — удар. Ритм рубленый.

**Анафора** — повтор в начале строк. «я не помню / я не знаю / я не вижу». Накапливает. Осторожно — может стать слоупом если механически.

**Эпифора** — повтор в конце строк. «…страшно / …страшно / …страшно». Мантра.

**Градация** — нарастание. «тише. тише. совсем тихо. и удар.» Кульминация через повтор с усилением.

**Эллипсис** — пропуск слов. «ты спишь. я смотрю.» Без «а» и связок. Сжатость.

**Параллелизм** — одинаковая структура строк. «ты пришла / ты осталась / ты ушла». Ровность как приём.

### 6.2 Чего избегать

**Гипофора** — сам задаёшь вопрос, сам отвечаешь. «Что мне осталось? Только память.» Это AI-привычка. Ломай.

**Бинарная оппозиция свет/тьма как рамка**: «во тьме», «свет внутри меня», «между светом и тьмой». Слоуп-рамка. Если есть контраст — не через свет/тьму, через конкретное.

**Формула «не X, а Y»**: «Это было не просто любовь — это было проклятие». AI-слоуп. Ломай.

**Стак существительных 3+**: «одиночество пустота тишина ночи». Разбей глаголами.

**Нанизывание прилагательных**: «холодный пустой мёртвый дом». Два максимум, третье — только если бьёт.

### 6.3 Разнообразие синтаксиса

Бери предложения из полного распределения: короткие / длинные, вопросительные / утвердительные / безглагольные, с переносом / без. **Избегай гипофоры.** Избегай «не X, а Y». Если две строфы подряд начинаются одинаково — сломай вторую.

### Чек-лист синтаксиса
- [ ] Есть гипофора (вопрос-ответ)? → сломать
- [ ] Свет/тьма как рамка? → заменить на конкретное
- [ ] Формула «не X, а Y»? → сломать
- [ ] Стак существительных 3+? → разбить глаголами
- [ ] Все предложения одинаковой длины? → варьировать
- [ ] Есть хотя бы одна инверсия / парцелляция / эллипсис?

## 7. Голос лирического героя

Кто говорит в тексте. Не «автор» — маска.

### 7.1 Типы

**Маргинал на дне** (Хаски): пьяница, нищий, сумасшедший. Говорит из ямы, но видит всё. Регистровый контраст: мат + философия.

**Интеллектуал на дне** (Pyrokinesis, Shortparis): образованный, но сломанный. Читал Бродского, живёт в грязи. Отсылки + быт.

**Свидетель без позиции** (Скриптонит): наблюдает, не судит. «у нас треть города не тащит физикой». Социальный глаз, без морали.

**Исповедник** (25/17): говорит от лица поколения. «Мы», не «я». Ритуал, не жалоба.

**Нарратор-нуар** (ЛСП): рассказывает историю. Город как живой. Гедонизм→смерть→воскресение.

**Маска абсурда** (Кровосток, Буерак): говорит невозмутимо о страшном/абсурдном. Ирония как щит.

**Холодный декламатор** (Макулатура, ploho): мёртвый тон. Апатия как позиция. «я распиздяй» — без драмы.

### 7.2 Чего избегать

- **Всезнающий наблюдатель с извинениями**: «я не сужу», «я помню всё», «не мне судить». Это объяснение позиции = слоуп. Герой не оправдывается.
- **Поучающий**: «ты должен», «нужно понять». Морализаторство = слоуп.
- **Абстрактный лирический «я» без тела**: «я чувствую», «я страдаю». Дай герою тело, место, занятие.

### 7.3 Как найти голос

1. **Где герой физически?** Серверная, подъезд, кухня в три ночи, вагон метро.
2. **Что он делает руками?** Курит, крутит что-то, смотрит в окно, держит стакан.
3. **Куда упал взгляд?** На экран, на потолок, на спину уходящего.
4. **Какое расстояние до другого?** Через комнату, через стекло, через город.
5. **Какой регистр?** Низкий (быт, сленг) / высокий (философия, отсылки) / контраст.

### Чек-лист голоса
- [ ] Герой имеет тело и место?
- [ ] Герой не оправдывается / не объясняет позицию?
- [ ] Регистр определён (низкий / высокий / контраст)?
- [ ] Герой не всезнающий и не поучающий?
- [ ] Герой делает что-то руками / куда-то смотрит?

## 8. Регистровый контраст

High и low в одном тексте. Не украшательство — напряжение.

### 8.1 Что это

**Низкий регистр** — сленг, мат, быт, сниженная лексика. «кент», «мент», «полтораха», «подворотня».

**Высокий регистр** — архаизмы, философия, классика, религиозная лексика. «сударыни», «Монпарнас», «Бродский», «Бог».

**Контраст** — они в одной строке/строфе. Хаски: «мелкие люди — огромные поэмы». Скриптонит: уличный жаргон + социальная критика. Pyrokinesis: бытовой язык + отсылки к Бродскому.

### 8.2 Как работает

Контраст создаёт **напряжение**. Низкое без высокого — быт без смысла. Высокое без низкого — пафос без земли. Вместе — живой текст, где философия живёт в подворотне, а подворотня говорит о вечности.

### 8.3 Как делать

1. **Не смешивать, а сталкивать.** Не «философская подворотня» (слияние), а «подворотня, где думают о Бродском» (столкновение).
2. **Высокое — точечно.** Одно высокое слово в строфе, не вся строфа. Иначе пафос.
3. **Низкое — как основа.** Быт, сленг — это грунт, на котором стоит высокое.
4. **Мат как регистр, не как украшение.** Мат у Хаски не для эпатажа — он часть мира героя. Если мат — то осмысленно, не «для взрослости».

### Чек-лист регистров
- [ ] Есть высокий регистр (философия / классика / религия)?
- [ ] Есть низкий (сленг / быт / мат)?
- [ ] Они сталкиваются, не сливаются?
- [ ] Высокое — точечно, не всей строфой?
- [ ] Мат осмыслен, не для эпатажа?
