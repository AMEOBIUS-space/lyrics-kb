## 4. Структура песни

Текст песни ≠ стихотворение. Песня имеет музыкальную форму, и текст должен в неё ложиться.

### 4.1 Базовые формы

**Verse-Chorus (куплет-припев)** — стандарт поп/рока:
```
[Verse 1]
[Chorus]
[Verse 2]
[Chorus]
[Bridge]
[Chorus]
```
Куплеты — развитие, припев — ядро. Припев один и тот же (или с вариациями).

**AABA** — классическая (джаз, Tin Pan Alley, частично поп):
```
[A] [A] [B] [A]
```
A — основная мысль, B — контраст («bridge» в старой терминологии). Плотная, закрытая форма.

**Сквозная (through-composed)** — без повторов, текст идёт от начала до конца (Макулатура, spoken word). Редко в песне, чаще в декламации.

**Циклическая** — один блок повторяется с вариациями (Shortparis «Страшно»). Рефрен-якорь вместо припева.

**Куплетная (строфическая)** — русская народная, шансон: все куплеты на одну мелодию, припева нет. «Ветер с моря дул».

### 4.2 Элементы структуры (meta-tags для Suno)

| Тег | Функция | Текст |
|---|---|---|
| `[Intro]` | вход, инструментал | обычно без текста, можно шёпот |
| `[Verse]` | куплет — развитие сюжета | основной текст, плотный |
| `[Pre-Chorus]` | подъём к припеву | короткий, нарастающий |
| `[Chorus]` | припев — ядро | хук, повтор, меньше строк |
| `[Bridge]` | контраст, смена | новая мысль, новый образ |
| `[Drop]` | электронная кульминация | минимум текста, повтор |
| `[Build]` | нарастание | короткие строки, ускорение |
| `[Climax]` | вершина | самый сильный образ |
| `[Hook]` / `[Catchy Hook]` | заедающий фрагмент | 1–2 строки |
| `[Instrumental Break]` | проход без голоса | пустота = музыка |
| `[Outro]` | выход | затухание, повтор ключевой фразы |

### 4.3 Динамика текста по форме

- **Verse** — плотный, развивает. Здесь можно много слов, рифм, образов.
- **Pre-Chorus** — сжатие. Меньше слов, больше напряжения.
- **Chorus** — ядро. Минимум слов, максимум удара. Хук.
- **Bridge** — перерыв. Другой ракурс, другой голос.
- **Outro** — затухание. Повтор ключевой фразы, вокализация.

Правило плотности: **от verse к chorus текст должен редеть, а не густеть**. Припев — не место для новых образов, а место для удара того, что накопилось в куплете.

### 4.4 Тишина как инструмент

Пустая строка. Пауза. `[Instrumental Break]`. Короткая строка после длинных. Это не пустота — это ритм. Самая сильная строка иногда та, которой нет.

### Чек-лист структуры
- [ ] Припев короче куплета?
- [ ] Хук в припеве — одна-две строки, не пять?
- [ ] Есть bridge (смена)?
- [ ] Текст редеет к припеву, не густеет?
- [ ] Есть хотя бы одна пауза (пустая строка / короткая после длинной)?

## 5. Образная система

### 5.1 Тропы (средства образности)

**Метафора** — перенос по сходству. «Панельное брюхо» — панелька как живот. Скрытое сравнение.
- **Свёрнутая** — короткая: «панельное брюхо».
- **Развёрнутая** — тянется через несколько строк/строф. «Сказки» Хаски — панелька как живой организм на весь трек.
- **Оксюморонная** — соединение несоединимого: «харчок золота» — уродливое в красивом.

**Метонимия** — перенос по смежности. «Серверная не спит» — стойки = люди. «Стакан» = выпивка.

**Синекдоха** — часть вместо целого. «Рука» = человек. «Колёса» = машина.

**Символ** — образ с устойчивым значением. Панелька = советский быт, упадок. Окно = выход, надежда (или её отсутствие).

**Олицетворение** — неживое как живое. «Серверная не спит», «панельное брюхо набито».

### 5.2 Образ-якорь и образ-триггер

**Образ-якорь** — повторяющийся образ, держащий трек. Хаски: «панелька» ×много. Shortparis: «хоровод». Якорь возвращает слушателя, накапливает смысл.

**Образ-триггер** — образ, создающий тревогу/настроение одним появлением. Shortparis: «потому и страшно». Не объясняет — констатирует. Триггер работает быстрее якоря, но слабее держит.

### 5.3 Как строить образ

1. **От конкретного.** Не «одиночество» — «холодный чай на столе в три ночи». Конкретный объект → абстрактное чувство.
2. **От тела.** Физическое действие несёт эмоцию. Не «мне больно» — «тяжесть по ступеням».
3. **От запаха/звука/фактуры.** Один специфический запах оживит строфу больше, чем пять абстракций. «Запах мокрого асфальта» сильнее «осенней тоски».
4. **Не объяснять образ.** Образ должен ударить, а не быть расшифрован. Если после образа идёт «это значит, что…» — сломать.
5. **Один образ, не три.** Нанизывание прилагательных (2+) — слоуп. Один бьющий образ сильнее.

### 5.4 Ризомная образность (Pyrokinesis)

Ризома — нелинейная сеть связей. Образы не выстроены в линию (А→Б→В), а соединены сетчато. Слушатель собирает смысл не с первого раза. Трек — не сообщение, а пространство.

Практически: в одном треке 4–5 ключевых образов, и они перекликаются между собой по разным линиям, не только последовательно. Один образ может отсылать к другому через 2 строфы.

### Чек-лист образов
- [ ] Образ конкретный (объект, тело, запах), не абстрактный?
- [ ] Образ не объясняется после?
- [ ] Есть образ-якорь (повтор)?
- [ ] Нет нанизывания 3+ прилагательных?
- [ ] Один бьющий образ на строфу, не три красивых?
