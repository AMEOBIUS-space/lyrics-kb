## 19. Разборы треков

Кейсы: от слоупа к живому.

### 19.1 Хаски — «Панелька» (эталон)

```
Начинается, как шум в животе.
Прятаться в мутной воде,
Вынырнуть в колыбельку
И врастать по-маленьку в панельку.

Прятки с отцом, горячо-холодно.
Жрать слёз маминых в поварешке.
Солнце тает в окне, как харчок золота.
Папа-папа был понарошку
В панельке (папа был понарошку!)

Набито панельное брюхо!
Панельного неба краюха!
Еду по России, не доеду до конца;
Где панелька моего отца.
```

**Что работает:**
- Тонический стих, удары 3-2-2-3, разная длина строк
- «жрать слёз маминых в поварешке» — телесный гротеск, тело пьёт слёзы
- «панельное брюхо» — панелька как живой организм (олицетворение + метафора)
- «харчок золота» — оксюморон, уродливое в красивом
- Повтор «панелька» / «папа» как рефрен-якорь
- Эстетика уродливого — панелька как герой, не фон
- Регистровый контраст: «поварешка» + «харчок золота» (бытовое + почти поэтическое)

### 19.2 ploho — «Я распиздяй» (думер)

```
Я распиздяй — такой мой жизненный путь,
Я пытался стать другим, но с дороги уже не свернуть.
```

**Что работает:**
- Мёртвый тон, самопризнание без драмы
- Простая рифма (путь—свернуть) как честность, не как лень
- Апатия как позиция
- Бытовая конкретика в языке, без усложнения

### 19.3 ploho — «Минорный ска»

```
Это минорный ска,
Очень грустная песня.
Пусто в моих карманах.
Соседка тупая овца...
```

**Что работает:**
- Разреженность, воздух между строками
- Бытовая конкретика: «пусто в карманах», «соседка овца»
- Слабая рифма — норма для жанра (пост-панк не требует)
- Холодный тон

### 19.4 Shortparis — «Страшно» (верлибр)

```
Потому и страшно
Встать в хоровод
Потому и страшно
```

**Что работает:**
- Одна фраза как мантра, накапливает тревогу
- Не объясняет причину — констатирует
- Рефрен-якорь вместо припева
- Верлибр, рифмы нет

### 19.5 Пошлая Молли — «Контракт» (поп-панк)

```
Я у Гвен Стефани украл её мотив
Чтобы ты сосала под мой эксклюзив
```

**Что работает:**
- AABB, точная банальная рифма — сознательно
- Ирония над «высокой поэзией» / шоу-бизом
- Прямолинейность = вирусность
- Мем-культура

### 19.6 Кровосток (звукопись)

```
Мы поднимем пожар на подошве бихевиоризма!
Бутылка: опустела. Перестало быть значимым вдоль
Мы заиграем пешка в шахматы вечного движения!
```

**Что работает:**
- Аллитерация (п-п-п-б-б) несёт ритм
- Абсурдная образность: «пожар на подошве бихевиоризма»
- Холодная констатация
- Философский сленг рядом с матом / бытовым

### 19.7 Кейс «серверная» — от слоупа к живому

**Слоуп (первая версия):**
- «слёзы в формате WAV» — эмоция в тех-обёртке = AI-ход
- «крик — в индекс / вызов — в лог» — лог-формат ×4, тик не музыка
- «помню всё» — клише
- «не сужу» — объяснение позиции
- Припев = чеклист показаний, не крюк
- «три-четырнадцать» ×3 — принудительный число-мотив

**Что сработало (последняя версия):**
- «серверная не спит» ×2 — рефрен-якорь, тянется голосом
- «ты спишь / я смотрю» — настоящее время, контраст, жуткое без объяснений
- «твой стук» повтор в бридже и аутро — музыкальный, не лог-структурный
- «ты вошла в три четырнадцать» — число один раз, как факт

**Чего всё ещё не хватает (по этому гайду):**
- Нет регистрового контраста — всё в одном нейтральном регистре
- Нет эстетики уродливого — серверная слишком абстрактна, нет запаха/фактуры/холода
- Нет фонетической работы — звуки не несут ритм
- Сленг слабый — «пробила», «в курсе» недостаточно
- Ризомности нет — линейный нарратив
- Нет физического якоря кроме «дыхание на лестнице»

**Как бы написал Хаски/Shortparis-смесь:**
- Серверная = конкретное уродливое пространство: гудящие стойки, жар кулеров, мёртвый свет лампы, кабельные стяжки
- Лирический герой — не абстрактный наблюдатель, а кто-то на дне (сисоп? охранник? сталкер?)
- Регистровый контраст: тех-жаргон + что-то высокое/религиозное
- Верлибр, рефрен-якорь

## 20. Чек-листы

### 20.1 Чек-лист «пройдёт ли это как живое»

1. Есть ли тех-метафора, оборачивающая эмоцию? (слёзы в WAV, сердце в архиве) → КРАСНЫЙ
2. Припев — чеклист или крюк? (статус-борд = слоуп)
3. Объясняешь ли ты позицию героя? («не сужу», «помню всё») → КРАСНЫЙ
4. Одинаковая длина строк? → подозрение на слоуп
5. Глагольные рифмы? → КРАСНЫЙ
6. Бинарная оппозиция свет/тьма как рамка? → КРАСНЫЙ
7. Гипофора (сам спросил — сам ответил)? → КРАСНЫЙ
8. Есть физический якорь — запах, тактильность, звук? → ЗЕЛЁНЫЙ
9. Регистровый контраст — сленг + высокое? → ЗЕЛЁНЫЙ
10. Рефрен-якорь вместо чеклиста? → ЗЕЛЁНЫЙ

### 20.2 Чек-лист перед отправкой (сводный)

**Метрика:** удары считаются, длина строк разная, есть перенос, есть удар (короткая после длинной)
**Рифма:** нет глагольных, нет банальных, 2–3 неточных, есть внутренняя, ради смысла
**Структура:** припев короче куплета, хук 1–2 строки, есть bridge, текст редеет к припеву, есть пауза
**Образы:** конкретные, не объясняются, есть якорь, нет нанизывания, один бьющий на строфу
**Синтаксис:** нет гипофоры, нет свет/тьма, нет «не X, а Y», нет стака существительных, варьируется
**Голос:** есть тело и место, не оправдывается, регистр определён, не поучает
**Регистры:** high и low сталкиваются, высокое точечно, мат осмыслен
**Уродливое/телесное:** конкретное пространство, тело не романтизировано, физический якорь, эмоция через тело
**Звукопись:** прочитано вслух, аллитерация в ключевой строке, звуки поддерживают настроение
**Интертекст:** отсылка работает на смысл, религия без пафоса, цитата точечная
**Хук:** 1–2 строки, заедает, не чеклист, в начале/конце
**Тёмное:** холодный тон, конкретика, тело, мат осмыслен, без морали
**Анти-AI:** слова-маркеры вырезаны, тех-метафор нет, конструкции сломаны, рифмы не банальные
