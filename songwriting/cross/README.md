# Songwriting — Cross (language-neutral)

Приёмы, общие для RU и EN. Не дублирует theory-ветки — только слой, который не ломается при смене языка.

---

## 1. Narrative arc of a song

| Beat | Job | Fail if |
|---|---|---|
| Open | place + body + itch | pure abstraction |
| Develop | one new fact/image | V2 = V1 paraphrase |
| Thesis | title/hook claim | thesis buried |
| Cost | what is lost and still visible | therapy comfort |
| Exit | action or residue | moral slogan |

**Rule:** listener can answer *where / who / what changed* after one pass.

---

## 2. Image engine

- One metaphor **family** per stretch (weather / room / machine / road / body)
- Vehicle filmable; tenor inferred
- Switch family only at section boundary with intent

See EN figurative + RU encyclopedia §5.

---

## 3. Topline / melodic phrasing (lyric side)

- Long notes → open vowels
- Clusters on short notes / offbeats
- Title on stable landing (downbeat or held note)
- Leave rests for breath at target BPM

Cross-read: `songwriting/en/theory/prosody.md`, RU §2/§13.

---

## 4. Section emotional dynamics

| Move | Lyric tool |
|---|---|
| Climb into chorus | unstable endings, rising phrase length |
| Chorus land | stable title, fewer words |
| Bridge valve | thinner, private fact |
| Bridge dive | denser confession |
| Final chorus | earned repeat + optional last-line twist |

---

## 5. POV / distance (language-agnostic)

- Pick person and stick unless marked shift
- Distance: close (body) / mid (scene) / far (report)
- Change distance when section job changes

---

## 6. Anti-slop universal

- Concrete > abstract
- Show cost, do not explain position
- No interchangeable checklist choruses
- No synonym-loop between V1 and V2
- Outro = fact/action, not consolation speech
