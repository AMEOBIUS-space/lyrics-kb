# Object writing — sense-bound timed drafts

Object writing (Pat Pattison) is a daily gym for concrete language. You pick an object and write for a fixed time using the senses — not about the object as symbol, but *through* it into place, body, and memory.

---

## 1. Why it works

Song clichés live in abstract nouns (*love, pain, forever*). Object writing forces:
- specific nouns
- active verbs
- sensory load the listener can film

It is not a finished lyric. It is raw ore for verses and bridges.

---

## 2. The seven senses

Go beyond five:

1. **Sight** — color, light, edge, motion
2. **Sound** — pitch, noise floor, rhythm in the room
3. **Smell** — the fastest memory trigger
4. **Taste** — literal or mouth-feel
5. **Touch** — temperature, texture, pressure
6. **Organic** (body interior) — breath, heartbeat, nausea, hunger, balance
7. **Kinesthetic** — movement through space, weight, acceleration

Aim to hit **at least four** in a 10-minute pass; all seven over a week of practice.

---

## 3. Protocol

1. Pick a random object (*bus ticket, chipped mug, motel key*).
2. Set a timer: **10 minutes** (beginners: 5).
3. Write without stopping. No rhyme goal. No song form.
4. Stay in **present tense** when possible.
5. When the timer ends — stop mid-sentence if needed.
6. Later: harvest 3–5 phrases that could survive in a lyric.

**Rules:**
- No explaining feelings by name if a sense can carry them
- No "it was beautiful" — show the glare on the chrome
- Allowed to drift from the object into scene — the object is a door, not a prison

---

## 4. Mini example (fragment)

Object: *motel key*

```
Plastic diamond on a cracked ring, number 214 worn to a ghost.
It smells like ashtray rain and someone else's shampoo.
The tooth of the key is warm from my fist. Vending machine
hums through the wall — a low sick note. Carpet grabs my
sock. I taste last night's coffee, burnt and sweet.
```

Harvest candidates: *number worn to a ghost*; *ashtray rain*; *vending machine… low sick note*.

---

## 5. From object writing → song

| Harvest | Where it often lands |
|---|---|
| One sharp image | Verse line 1–2 (hook into world) |
| Body/organic line | Bridge or pre-chorus vulnerability |
| Smell + place pair | Title orbit / recurring motif |
| Motion line | Transition into chorus |

Don't paste a whole paragraph into a chorus. Compress.
