# Boxes / worksheets — from title to draft

"Boxes" are constrained worksheets that expand a title into angles before you write linear lyrics. Used in Pattison-style rooms and many commercial co-writes.

---

## 1. Why boxes beat freewrite-first

Freewrite finds voice. Boxes find **structure of meaning** so verse 2 isn't verse 1 in a wig.

---

## 2. Title box (start here)

Write the working title in the center. Around it, fill:

| Box | Prompt |
|---|---|
| **Literal** | What does the title mean on the nose? |
| **Ironic** | When is the title a lie or half-true? |
| **Image** | One filmable picture the title suggests |
| **Opposite** | Antonym world (if title is "Stay", explore leaving) |
| **Who says it** | Character / age / power position |
| **Who hears it** | Addressee |
| **Time** | When in the story is this said? |
| **Cost** | What does believing the title cost? |

Pick **one primary angle** + **one secondary** before drafting.

---

## 3. Sense box (per section)

For Verse 1 / Verse 2 / Bridge — three columns:

```
SEE          HEAR         BODY
-----------  -----------  -----------

```

Fill 3 cells each. Steal the best cell into the draft; delete the rest.

---

## 4. Question box (story engine)

List 8 questions the song could answer. Star only 3.

Examples for title *"Don't Wait Up":*
- Who usually waits?
- What broke the pattern tonight?
- Is it kindness or cruelty?
- What does the house do when empty?

Verses answer questions. Chorus states the title-claim. Bridge answers the question the chorus avoids.

---

## 5. Rhyme / sound box (optional, late)

Only after meaning is sketched:
- target end-words for chorus
- family-rhyme lists (avoid cliché pairs first — see `../theory/rhyme.md`)
- vowel colors for the title note

---

## 6. Contrast box (verse vs chorus)

| Axis | Verse | Chorus |
|---|---|---|
| Tense | | |
| POV | | |
| Word count feel | denser / sparser | |
- Rhyme tightness | looser / tighter | |
| Image type | specific / general claim | |

If two columns match, force a difference before writing.

---

## 7. 30-minute co-write sequence

1. Title box — 5 min  
2. Question box — 5 min  
3. Sense box V1 — 5 min  
4. Draft chorus (title stable) — 7 min  
5. Draft V1 from sense box — 5 min  
6. Mark prosody stresses — 3 min  

---

## 8. Checklist

- [ ] Title has a chosen angle (not all angles at once)?
- [ ] V1 and V2 have different answered questions?
- [ ] Sense boxes produced at least one non-visual sense?
- [ ] Contrast box shows real section difference?
