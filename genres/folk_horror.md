# Жанровый плейбук: Фолк-хоррор

Полноценный жанровый стандарт по образцу общего жанрового плейбука. Работает поверх энциклопедии (§25–28 v2.0). Жанр выделен по итогам разведки 2025–2026 как самый живой тренд сцены.

[источник: Notion, 2026-07-11]

---

## 1. Вайб и рамка жанра

- Фолк-хоррор — это не «песни про деревню». Это взгляд, при котором знакомый быт (панелька, огород, трасса, лесополоса) оказывается населён чем-то древним и недобрым — или наоборот, жалостливым и родным.
- Формула вайба: **быт + миф = жуть**. Одна нога в документальной конкретике, вторая — в заговоре, причете, быличке.
- Эмоциональный диапазон: тоска, жуть, умиление вперемешку. Запрещено: чистый ужас без нежности и чистая ностальгия без жути.
- Референсы: **ATL** (бестиарий окраин), **Хаски** (плоть и молитва), **Букер** (башлачёвские колокольчики), **Бонд с кнопкой** (глоссолалия + кухни), **Аффинаж**, **25/17**; поэты-доноры: Башлачёв, Клюев, ранний Есенин, Некрасов (причетная интонация), Шварц (юродская мистика).

## 2. Голоса жанра (из галереи §7 + специфика)

- **свидетель** — базовый: видел нечто, рассказывает без оценок, детали точные.
- **причитающий** (новый подвид): говорит с мёртвым/ушедшим/домом как с живым, формулы повтора и обращения («ты куда же, ты зачем же»).
- **юродивый** (маска_абсурда в фолк-изводе): говорит правду через нелепицу, скоморошья интонация при страшном содержании.
- Запрещено: интеллигент-этнограф (взгляд сверху на «народ» = лубок и фальшь).

## 3. Лексика: два словаря в одном тексте

```
СЛОВАРЬ А (быт, документ): гараж, теплица, сетка-рабица, электричка,
  баллон газа, комбикорм, ватник, шифер, колонка (вода), сельмаг
СЛОВАРЬ Б (миф, заговор): кровь-руда, навь, межа, оказия, сглаз,
  отчитать, некрещёный, помин, кутья, оберег
ПРАВИЛО СМЕШЕНИЯ: в каждой строфе — минимум одно слово из каждого словаря.
  Миф без быта = косплей неофолка. Быт без мифа = просто бытопись.
ЗАПРЕЩЕНО (лубок): берёзка, косоворотка, «русь матушка», балалайка,
  «широкая душа», кокошник — кроме как в иронии/сломе
```

## 4. Просодия и метрика

- База: **тоника и дольник** (§2), считаем удары, не слоги. Фолклорная основа — трёхударная строка причета.
- Рабочий приём: переключение метра между куплетом (говорной дольник) и припевом (чёткая фолклорная формула, как считалка или заговор).
- Синтаксические фолк-маркеры (дозированно, 1–2 на текст): инверсия («воды тёмной набери»), постпозитивное прилагательное («трава-мурава»), парные слова («жили-были», «путь-дорога»).
- Пропеваемость (P11): припевные строки кончать на открытые гласные а/о/у — под протяжное пение.

## 5. Рифма

- Допустимо всё от точной народной (частушечной) до безрифменного причета. Главное — осознанность выбора на уровне секции.
- Фирменный ход жанра: рифма бытового с сакральным («кутья / станция Кучино моя», «оберег / сбер») — регистровый контраст §8 на уровне рифмопары.
- Запрещено: глагольные пары без компенсации (verb_rhyme из §25).

## 6. Структура

- Рабочие формы: циклическая (заговор по кругу), куплет-припев с припевом-формулой, сквозная быличка (рассказ о встрече с необъяснимым без развязки).
- Припев-формула: строится как заговор/считалка — повтор синтаксической рамки с заменой одного слова. Это НЕ chorus_checklist: формульность здесь жанровая конвенция, но заменяемое слово обязано копить смысл (как образ-якорь §5).
- Финал: открытый или цикличный (вернуться к первой строке с изменившимся смыслом). Запрещено: вывод-мораль (school_arc).

## 7. Образность: мифологемы жанра

```
МЕХАНИКА: взять бытовой объект → дать ему фолклорную функцию
ПРИМЕРЫ:
  лесополоса = межа между мирами
  гаражи = капище (мужики собираются и молчат о главном)
  электричка = лодка Харона (увозит и не возвращает)
  теплица = мавзолей лета
  старый холодильник в овраге = идол, которому больше не носят дары
ПРАВИЛО: мифологема работает, только если объект описан документально
  (цвет, ржавчина, звук) — миф должен проступать сквозь фактуру, а не заменять её
```
- Приём «умолчание» (разведка §3.4) — штатный инструмент жанра: жуть сильнее, когда событие не названо.

## 9. Анти-слоп чек-лист жанра

- [ ] В каждой строфе есть слово из словаря А (быт) и словаря Б (миф)
- [ ] Нет лубка (берёзки-балалайки) без иронического слома
- [ ] Главная жуть не названа прямо — только следы
- [ ] Мифологема опирается на документальную фактуру (можно сфотографировать)
- [ ] Припев-формула копит смысл, а не просто повторяется
- [ ] Есть хотя бы одна строка умиления/нежности внутри жути (и наоборот)
- [ ] Прогнано через Детектор 2.0 + оценка с жанровыми весами «неофолк» (до появления отдельного профиля)

## 11. Мини-эталон (калибровочная строфа)

```
за сеткой-рабицей ботва да ботва́,
бабка говорила: за овраг — ни ногой.
бабки нет три года. теплица жива:
кто-то поливает там по ночам огурцы.

разбор: словарь А (сетка-рабица, теплица, огурцы) + Б (запрет межи «за овраг ни ногой»);
жуть через умолчание (кто поливает?); нежность (бабка) внутри жути;
финал без вывода — только факт.
```
