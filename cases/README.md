## Series lessons (apply to every new CW gen)

- Pre-chorus **required**; chorus **×2** back-to-back (from CW-003)
- No Mm-intro — plot intro only (CW-004)
- INST before bridge: two-phase preferred; plan B = 4-bar single phase (CW-004)
- No timestamp gimmick after CW-005
- Style = tag format + `professional studio recording` (CW-005)
- Anti-wail: `[No Vocals][No Humming][No Wailing][No Vocal Ad-Libs]` on intro/INST
- Default coldwave EQ stack: low shelf −2.5@100 · high shelf +4@3k · high shelf +4@10k · limiter −0.5 dBTP
- mud_2k watch: bell −1.5@300 (CW-002)
- ≥2 Style slots change per track; core: darksynth/coldwave + deep husky baritone (until explicit POV break)
