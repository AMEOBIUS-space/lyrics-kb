## Как использовать

### Mood → Sound (INVERSE_MAP расширение)
Топ-10 moods покрывают ~60% всех артистов. Добавь в генератор стиля:
- melancholic → minor key, breathy/intimate vocals, fingerpicked, warm pads
- theatrical → dramatic dynamics, powerful tenor/soprano, orchestral, reverb tail
- nostalgic → analog synths, tape saturation, retro production, era-specific
- anthemic → double-tracked choruses, stadium crowd, soaring belt, big drums
- swaggering → confident mid-tempo, prominent bass, crisp delivery, punch
- intimate → close-mic, breathy, minimal instrumentation, dry, no compression

### Use case → Energy/Tempo
- nostalgic-listen → mid-tempo 90–110, warm, no aggression
- gym-pump → 128–145, driving, aggressive, four-on-the-floor
- study-focus → 80–100, instrumental, lo-fi, no vocals, subtle
- party → 115–130, energetic, catchy hooks, danceable
- late-night → 85–105, moody, intimate, atmospheric
