# Процедура: фикс пропадающих инструментов в припеве Suno

## Симптом
В припеве инструменты пропадают, остаются только ударные и паулы. Особенно в латино, реггетоне, афробите, танцевальной поп-музыке.

## Причина
Suno воспринимает припев как «место для вокала» и вырезает инструменты.

## Шаг 1: Добавь в STYLE
```
non-stop rhythm section, constant bass groove, percussion stays active during chorus, full instrumental support behind vocals
```

## Шаг 2: Добавь теги плотности
Для латино/танца используй:
- ✅ `club anthem`
- ✅ `festival latin pop`
- ✅ `dancefloor chorus`
- ✅ `full groove chorus`

## Шаг 3: Убери теги разреженности
НЕ используй:
- ❌ `emotional`
- ❌ `cinematic`
- ❌ `dramatic`

## Шаг 4: Варьируй длину строк
Если куплет и припев написаны одинаково длинными строками → Suno освобождает место под вокал и вырезает инструменты.

Решение: сделай строки припева короче или длиннее чем в куплете.

## Шаг 5: Добавь структурные метки в лирику
```
[Chorus]
[Non-Stop Rhythm]
[Full Instrumental]
```

## Контрольный чеклист
- [ ] `non-stop rhythm section` в style
- [ ] Теги плотности добавлены
- [ ] Теги разреженности убраны
- [ ] Длина строк варьируется между секциями
- [ ] Структурные метки в лирике
