# Процедура: ровный рок-вокал в Suno без визга

## Шаг 1: Выбери жанр
Используй один из: **Melodic Rock, Blues Rock, Classic Rock, Soft Hard Rock, AOR, Heartland Rock**
НЕ используй: Hard Rock, Heavy Metal, Arena Rock — они тянут вокал в крик.

## Шаг 2: Style Prompt — вставь этот блок целиком
```
controlled rock vocal, warm chest voice, mid-range male vocal, expressive but restrained delivery, no screaming, no shouting, no high-pitched vocals, no aggressive belting, controlled emotional intensity, melodic rock singing, stable vocal tone, mature rock voice
```

## Шаг 3: Ключевая фраза
Добавь `no vocal acrobatics` — одна фраза убирает половину ненужных верхов.

## Шаг 4: Дублируй ограничения в лирике
Перед припевом вставь метки:
```
[Controlled Rock Vocal]
[Mid Register Only]
[Warm Chest Voice]
[No Scream]
[No High Belt]
[Controlled Emotional Delivery]
```

## Шаг 5: Опусти BPM
Снизь BPM на 5-10 единиц от задуманного → Suno поёт спокойнее.
Медленный грув = менее агрессивная подача.

## Шаг 6: Выбери версию
- v4: чище вокал, лучше структура, стабильнее
- v3.5: сырее, рванее — иногда лучше для рока
- v4.5: самый чистый контроль (июнь 2026)

## Шаг 7: Референсы для калибровки
Дэвид Ковердейл (спокойные вещи), поздний Иэн Гиллан, Марк Нопфлер, Крис Ри, рок-баллады 80-х.

## Что НЕ работает
- `powerful rock vocal` → визг
- `aggressive vocal` → крик
- `screaming vocal` → карикатура
- `arena rock vocal` → оркестровая попса
- `soaring vocal` → верхний регистр

## Контрольный style tag для финальной генерации
```
mature male rock vocal, chest voice focused, emotional restraint, melodic phrasing, no vocal acrobatics
```
