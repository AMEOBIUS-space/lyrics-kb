# EN cliché rhyme hotlist

Flag as **E08** unless whitelist W2 (irony) applies.

## Tier A — almost always lazy

- fire / desire / higher / liar
- heart / apart / start / dark
- love / above / of *(sung)* / enough
- night / light / right / fight / tight
- eyes / skies / lies / cry / why
- together / forever
- rain / pain / again / remain
- home / alone / own
- true / you / blue / through
- world / girl
- free / me / see / destiny
- go / know / show / alone *(when chained)*
- day / way / say / away
- believe / leave / sleeves
- touch / much
- late / wait / fate
- scar / far / are
- dream / seem / machine *(pop default)*
- broken / open
- falling / calling

## Tier B — tired in pop/country

- truck / luck / dust
- beer / here / year
- road / home / alone
- small town / run around
- neon / feeling
- whiskey / miss me
- midnight / highlight / streetlight

## Tier C — EDM / big-room vocal

- higher / fire / alive / tonight
- shine / time / life / light
- forever / together / better
- rise / skies / life

## Repair recipes

1. Keep the **non-title** word; replace its partner with a concrete unexpected noun.  
   *heart / apart* → *heart / parking ticket*
2. Switch to **family rhyme** or assonance.  
   *night / light* → *night / like* / *night / silence*
3. Move the rhyme **inside** the line; end on an image unrhymed.
4. Drop end-rhyme for one couplet; hold with rhythm only.
