# Продакшен: мастеринг

## LUFS-таргеты по платформам
- Spotify: -14 LUFS
- Apple Music: -16 LUFS
- YouTube: -14 LUFS
- Bandcamp: не оптимизирует (name your price, direct to listener) — можно громко
- Tidal: -14 LUFS

## True peak
- Лимит -1 дБ true peak для стриминга
- True peak +1 и выше = клиппинг, режет на хороших мониторах
- Для Bandcamp/релиза: можно допускать намеренную грязь (garage ethos)

## Эталон по фактуре (ПЕРЕМОЛОТ как пример намеренной грязи)
- LUFS -6.4 до -9.9 — громко для фолка
- LRA 5.4-7.6 — средняя динамика, не кирпич
- True peak +2.5/+2.8 — клиппинг, но это осознанный выбор
- Тёмный, вокал-форвардный, верха завалены (-25 дБ на 6k+)

## Multiband
- Для проблемного материала
- Низ / низ-мид / мид / верх — отдельно

## Лимитер
- На выходе
- Ceiling -1 дБ для стриминга
- Для Bandcamp — можно без (если осознанная грязь)
